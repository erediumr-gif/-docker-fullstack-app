# Docker Full Stack App

## Tech Stack

* Docker & Docker Compose
* Nginx (Reverse Proxy)
* Node.js (Backend API)
* MongoDB (Database)

## Features

* Multi-container application
* Reverse proxy using Nginx
* REST API with Node.js
* Persistent storage using Docker volumes

## Run Project

```bash
docker-compose up -d
```

## Access

http://<your-ec2-ip>
